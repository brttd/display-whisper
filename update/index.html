Updating Display Whisper

Downloads for latest version: Windows To use, download & extract the zip. Then run "Display Whisper.exe" from inside the extracted folder. (Installer coming in the future) MacOs To use, download & extract the zip. Then Control-click on the extracted app, and choose "Open" from the menu. (Signed version coming in the future) Library & Settings are stored outside of program location, so any old versions can safely be deleted. For more information, view the GitHub page.
